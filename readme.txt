Adds a NRIC/FIN field type to Advanced Custom Fields plugin.

== Description ==

This field validates NRIC by default and FIN inputs when "Allow FIN" setting is set to true. This plugin requires the Advanced Custom Fields plugin.

= Compatibility =

This ACF field type is compatible with:
* ACF 5 and up
* ACF 4 (not supported)

= Credits =

Thanks to Sam-Mauris Yong for the NRIC Utility script.

== Installation ==

1. Copy the `acf-nric` folder into your `wp-content/plugins` folder
2. Activate the NRIC/FIN plugin via the plugins admin page
3. Create a new field via ACF and select the NRIC/FIN type
4. Please refer to the description for more info regarding the field type settings
